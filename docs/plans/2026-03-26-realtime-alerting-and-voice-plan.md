# 足球实时提醒与语音播报专项

> 更新时间：2026-03-26  
> 适用范围：足球实时盯盘助手、Feishu 提醒、本地语音播报

## 1. 产品定义

最终产品不是离线报告器，而是实时盯盘助手：

- 跟着人一起看直播
- 关键事件出现时及时提醒
- 给一句短方向建议
- 在关键窗口做语音播报

## 2. 第一版状态机

- `watch`
- `enter_first_leg`
- `hedge_ready`
- `invalidated`

## 3. 第一版提醒输出

- 事件类型 / 触发家族
- 建议关注方向
- 当前是否进入窗口
- 预计多久关注反手
- 简短理由
- `voice_text`

## 4. 语音原则

- 只播关键提醒
- 不做全程解说
- 一次只播 1-2 句
- 文案要短、明确、可执行

## 5. 对冲计算原则

第一版只做计算和展示，不自动下单。
