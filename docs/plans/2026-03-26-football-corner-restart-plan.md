# 足球角球 / 重启识别专项

> 更新时间：2026-03-26  
> 适用范围：足球实时观察员主线 B、角球提醒、重启规则映射

## 1. 目标

这条线不是做通用足球裁判，而是做对交易有价值的重启事件识别：

- 球是否已经出界
- 是边线还是底线
- 最后触球方是谁
- 这次重启是界外球、球门球还是角球
- 是否值得触发角球观察提醒

## 2. 第一版只做什么

- `ball_out_of_play_detected`
- `exit_boundary`
- `last_touch_side`
- `restart_type`
- `corner_candidate`

第一版不做：

- 越位
- 手球争议
- 任意球复杂判罚
- 端到端“模型直接裁判所有规则”

## 3. 系统原则

- 视觉模型负责提取事实
- 规则层负责映射足球规则
- 角球提醒必须保守，宁可漏掉也不要大量误报
- 角球线最终输出到统一提醒 contract，不单独起一套交易层
